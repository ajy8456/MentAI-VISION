✨ Overview

'나의마음건강확인하기'는 사용자의 현재 감정 상태와 스트레스 지수를 과학적인 데이터를 기반으로 분석해주는 혁신적인 서비스입니다. 언제 어디서나 간편하게 자신의 마음 건강을 체크하고 맞춤형 솔루션을 제공받으세요.

🎬 홍보 영상

서비스의 핵심 가치를 담은 홍보 영상을 만나보세요.

주요 가능

'나의마음건강확인하기'는 다음과 같은 핵심 기능들을 제공합니다.

  • AI 기반 감정 분석: 사용자의 텍스트, 음성, 표정 데이터를 복합적으로 분석하여 현재 감정 상태를 진단합니다.
  • 일일 스트레스 리포트: 매일의 활동 패턴과 응답을 기반으로 스트레스 지수를 시각화하여 리포트로 제공합니다.
  • 맞춤형 힐링 콘텐츠 추천: 분석 결과에 따라 명상, 음악, 아티클 등 개인에게 최적화된 힐링 콘텐츠를 추천합니다.
  • 전문가 연계 (선택사항): 필요시 검증된 심리 상담 전문가와의 비대면 상담을 연결해 드립니다.

서비스 시연 영상

'나의마음건강확인하기' 서비스의 실제 작동 과정을 시연 영상을 통해 확인해보세요. 앱 실행부터 감정 분석 결과 확인까지의 전 과정을 상세하게 담았습니다.

영상을 보시면, 사용자가 어떻게 텍스트를 입력하고, AI가 이를 분석하여 어떤 피드백을 주는지 명확하게 이해하실 수 있을 것입니다. 간편한 인터페이스와 신속한 분석 속도가 본 서비스의 강점입니다.

  • 모바일 환경에 최적화된 UX
  • 간단한 입력으로 즉시 분석
  • 직관적인 결과 시각화

기술 설명

본 서비스는 최신 **자연어 처리(NLP)** 기술과 **컴퓨터 비전** 기술을 활용한 복합 AI 모델을 기반으로 합니다. 사용자 데이터는 **AES-256 암호화**를 통해 안전하게 관리되며, 머신러닝 알고리즘이 지속적으로 학습하여 분석의 정확도를 높여나갑니다.

  • Frontend: HTML, CSS, JavaScript (React/Vue 등)
  • Backend: Python (Django/Flask), Node.js
  • AI/ML: TensorFlow, PyTorch, Scikit-learn
  • Database: PostgreSQL, MongoDB
  • Cloud: AWS / Google Cloud Platform
